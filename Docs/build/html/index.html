Web Radio Documentation¶

Web Radio is a project developed by LabIoT Team in Parque Tecnológico de Itaipu (PTI) under the SmartOffice project. The goals is to create a personalized radio via internet for offices. The radio server is based on Liquidsoap and Icecast. The radio client can be any device that can play a stream from an URL, but here you will find a Python script that uses the VLC library in Python wrapper to capture the HTTP packages and play the audio.

Note

This project is under active development and comes with no garanties.
